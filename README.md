# GENTLE MONSTER AI STUDIO

**VERSION 1.1 (OFFICIAL RELEASE)**

A hyper-advanced, web-based visual programming environment powered by Google Gemini AI. Designed exclusively for the Gentle Monster creative team to generate, render, and manipulate spatial and visual concepts through natural language and generative AI.

---

## OVERVIEW

This platform bridges the gap between abstract creative direction and tangible visual output. It leverages the latest in generative AI to allow rapid prototyping of avant-garde concepts, ensuring seamless integration with the Gentle Monster aesthetic.

### Key Capabilities
*   **Generative AI Integration**: Direct interface with Google's Gemini models for high-fidelity image generation.
*   **Visual Programming**: Spatial and visual manipulation of generated assets.
*   **Secure Environment**: Protected access for internal Space Team use only.
*   **Cross-Platform Core**: Optimized `gm-loader` engine for consistent performance across desktop and mobile interfaces.

---

## USAGE GUIDE

### Access
*   **Authentication**: Restricted access via the "Space Team" password gateway.
*   **API Configuration**: Requires a valid Google Gemini API Key.
    *   Keys can be generated directly via the usage interface ("GET API KEY").

### Workflow
1.  **Authenticate**: Enter credentials at the secure gateway.
2.  **Connect**: Input your API Key to initialize the AI core.
3.  **Select Mode**: Choose between **CREATE** (Generative Phase) or **RENDER** (Processing Phase).
4.  **Execute**: Input prompts or upload references to begin visual programming.

---

## SYSTEM ARCHITECTURE

*   **Core Engine**: `assets/gm-loader.js` (v2.0)
    *   Handles authentication state, mobile responsiveness, and dynamic UI injection.
    *   Ensures stability by managing DOM injection timing and race conditions.
*   **Frontend**: React-based SPA (Single Page Application) compiled for performance.
*   **Deployment**: Automated deployment pipeline via Vercel/GitHub.
